Remote, Gesture and Voice controlled chassis 2023

Designed and developed a multi-mode wireless controlled chassis powered by Arduino and Bluetooth technology. Integrated a Bluetooth module with custom Arduino code to enable control via mobile app, supporting three distinct input modes: traditional remote, voice commands (audio), and gesture-based controls. The system featured real-time responsiveness and intuitive switching between control methods, enhancing user interaction. During the testing phase, the optimized input handling and gesture recognition system reduced user input time by approximately 40%, significantly improving operational efficiency and usability.
